### 💼 AI‑команда для маркетинговой стратегии (CrewAI + Proxy API)

Проект создаёт **команду ИИ‑агентов** на базе **CrewAI**, которая генерирует маркетинговую стратегию в несколько этапов:

1. Исследование рынка
2. Бренд‑стратегия и позиционирование
3. 90‑дневный план продвижения (roadmap, KPI, эксперименты)

Работает через **OpenAI‑совместимый Proxy API**

---

## Что умеет

- Генерирует стратегию на **русском языке**.
- Подходит для разных ниш (B2B/B2C) — достаточно описать продукт, аудиторию, географию и бюджет.
- Есть **веб‑интерфейс** (форма в браузере) и **консольный режим**.
- Экспорт результата:
  - скачать стратегию в `.md`
  - скачать стратегию в `.docx` (Word)
